About the Project

VIN Traffic System brings together realistic traffic simulation, machine‑learning prediction, and an operator‑friendly dashboard. It supports multiple Indian cities, highlights congestion hotspots, and provides actionable insights through clear KPIs, charts, and a live map.

  • Professional Dashboard: Clean, responsive UI with tabs for Monitor, Analytics, Map, and Data.
  • Realtime Simulation: Streamed samples with auto‑refresh, alerting, and status monitoring.
  • Analytics: Speed/volume distribution and trends with rolling metrics and summaries.
  • Map View: City heat overlays by alert level using pydeck/Deck.gl.
  • Exports: One‑click CSV downloads for reports and offline analysis.

Key Features

Realtime

Live data collection with configurable refresh interval and status pill.

ML Prediction

Congestion prediction using scikit‑learn models and alerting rules.

Analytics

Smart detection of speed or volume across datasets; robust charts.

Map

City map with colored scatter layers by alert severity and tooltips.

Exports

CSV download for both live and processed datasets.

Dockerized

Ready‑to‑run container images for quick demos and deployments.
